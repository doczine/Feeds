Azure backups are stored to Azure storage. </p>  <p>To begin, you must have an Azure Storage account. If you have not yet created one, see <a  href="http://davidgiard.com/ct.ashx?id=22554245-c795-4cfb-8dff-b146d7dd49e0&amp;url=http%3a%2f%2fdavidgiard.com%2f2015%2f04%2f25%2fCreatingAnAzureStorageAccount.aspx" target="_blank">my post on creating a new Azure Storage Account</a>. </p>  <p>Navigate to your Web App by selecting the WEB APPS icon in the left menu; then, selecting your Web App. </p>  <p>Note that these features are only available if your Web App is using the STANDARD Pricing Plan Tier. You can modify your Pricing Plan Tier on the SCALE tab of your Web App. </p>  <p>Click the BACKUPS tab (Figure 1)of your web app to start managing your backups. </p>  <p><a  href="http://davidgiard.com/ct.ashx?id=22554245-c795-4cfb-8dff-b146d7dd49e0&amp;url=http%3a%2f%2fdavidgiard.com%2fcontent%2fbinary%2fWindowsLiveWriter%2fBackingUpaWebApp_39FE%2fBackup01-BackupTab.png"><img title="Backup01-BackupTab" style="border-left-width: 0px; border-right-width: 0px; border-bottom-width: 0px; display: inline; border-top-width: 0px" border="0" alt="Backup01-BackupTab" src="http://davidgiard.com/content/binary/WindowsLiveWriter/BackingUpaWebApp_39FE/Backup01-BackupTab_thumb.png" width="244" height="215" /></a>     <br />Figure 1 </p>  <h3>Manual Backup</h3>  <p>To Manually backup your Web App, do the following </p>  <p>1.
